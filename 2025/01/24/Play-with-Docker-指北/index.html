启动 Ubuntu 并安装 Neofetch

启动一个带有特权模式的 Ubuntu 容器（注意：在本地环境中不要这样操作，因为特权模式的容器对宿主机的文件系统有较高的访问权限，存在安全风险），并将其挂载到主机文件系统：

  • -itd 让容器在后台运行但仍然保持交互性。
  • --privileged 给予容器额外的权限。
  • -v /:/mnt 将宿主机的根目录挂载到容器内的 /mnt 目录。
  • --network host 使用宿主机的网络栈。

Vim 快速指北（指南）

vim 的操作方式与大多数现代编辑器不同。

启动和退出

  • 启动：命令行中输入 vim 文件名（下文用到此操作）或 vim。
  • 退出（在普通模式下执行）：
    • 不保存更改退出：:q! 并按下 Enter 键
    • 保存并退出：:wq 或 ZZ 并按下 Enter 键（下文用到）

模式

Vim 主要有（下文用到）三种模式：

  1. 普通模式：默认模式，用于执行命令，在其他模式中可通过按 Esc 返回普通模式。
  2. 插入模式：按下 i 进入此模式，开始编辑文本。
  3. 命令行模式：按 : 进入此模式，输入命令如保存文件或退出。

demo_1 编写 C++

  1. 在 home 目录创建一个 cpp 文件夹并在此目录下新建 C++ 文件 hello.cpp。
  2. 安装编译器 g++：apt install g++ -y（在 vscode web 界面打开新的终端执行此命令）。
  3. 转到 cpp 目录下：cd cpp, 编译代码：g++ -o hello hello.cpp。
  4. 运行程序：./hello。

demo_2 Web 开发

  1. 创建 web 文件夹，并在其中建立 HTML 文件 index.html。
  2. 使用 Emmet 插件快速生成 HTML 结构：输入 ! 并按 Enter。
  3. 安装 Bun：curl -fsSL https://bun.sh/install | bash，然后执行： source ~/.bashrc。
  4. 转到 web 文件夹, 安装并指定端口启动： bunx http-server --port 80
  5. 浏览器访问该网页。
  6. 最后可以使用 ctrl+c 结束 http-server 进程。
